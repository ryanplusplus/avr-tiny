# avr-tiny
## Setup
```shell
make -f target.mk install_toolchain
```

## Use
### Compile
```shell
make -f target.mk
```

### Clean
```shell
make -f target.mk clean
```

### Erase via selected tool
```shell
make -f target.mk erase
```

### Flash via selected tool
```shell
make -f target.mk upload
```
